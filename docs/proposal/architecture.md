# 잇다서울 아키텍처 메모

## 프론트엔드

- React + TypeScript + Vite
- 사용자 상태 입력, 추천 결과, 공유 카드, 기관 화면 프로토타입

## 백엔드

- Node.js + Express + TypeScript
- 추천 API
- 데이터 정규화 API
- 향후 AI 설명 생성 모듈 연결

## 데이터 저장

- PostgreSQL
- 사용자 프로필, 추천 이력, 관심사, 참여 이력 저장

## AI 역할

- 사용자 입력 해석
- 외출 가능성 및 연결 우선순위 분류
- 추천 이유 설명
- 보호자와 복지사에게 전달할 문구 생성
