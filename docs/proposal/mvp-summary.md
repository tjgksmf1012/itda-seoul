# 잇다서울 MVP 요약

## 한 줄 정의

잇다서울은 서울시 공공데이터를 이용해 고립위험 1인가구와 고령층에게 지금 당장 필요한 다음 외출, 다음 참여, 다음 연결 행동을 추천하는 AI 사회연결 처방 플랫폼이다.

## 문제 정의

- 정보는 많지만 행동으로 이어지지 않는다.
- 고립 문제는 검색의 부재보다 연결 실패의 문제다.
- 보호자와 복지현장은 프로그램 탐색, 링크 전달, 참여 확인을 반복 수작업으로 수행하고 있다.

## MVP 범위

- 사용자 상태 입력 화면
- AI 추천 결과 화면
- 공공서비스예약 링크 연결
- 보호자 공유 메시지 생성
- 기관용 추천 관리 화면 초안
- 기관용 상태 변경과 추천 이력 관리

## 데모 포인트

- 단순 목록 나열이 아니라 다음 행동 1개를 추천한다.
- 추천과 동시에 신청 링크와 공유 문구를 생성한다.
- 고립위험군, 보호자, 복지현장 각각의 대표 시나리오를 보여준다.
- 기관 담당자가 대상자 카드를 선택하면 개인 추천기와 바로 연동된다.
- 추천 결과에 실행 체크리스트, 매칭 점수, 후속 체크인 문구를 함께 제공한다.
