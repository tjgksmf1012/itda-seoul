# 잇다서울 데이터 소스 정리

## 필수 데이터

- 독거노인 현황
- 노인여가복지시설
- 공공서비스예약 정보
- 문화행사 정보
- 예약형 문화행사 정보

## 공식 참고 페이지

- 서울 열린데이터광장 Open API 현황
- 서울시 문화행사 정보
- 서울 공공서비스예약
- 서울시 공공데이터 목록의 노인여가복지시설 관련 데이터셋

## 활용 목적

- 고립위험 지역과 사용자 타깃 설정
- 오프라인 연결 거점 확보
- 추천 이후 즉시 신청 가능한 행동 연결
- 취향과 접근성에 맞는 외출 후보 생성

## 정리 원칙

- 종료되었거나 만료된 행사 자동 제외
- 갱신일 기준으로 최신성 확인
- 위치, 비용, 신청 가능 여부를 공통 스키마로 정규화
