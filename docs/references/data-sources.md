# 오늘서울 데이터 소스 정리

## 1. 서울 열린데이터광장

역할: 시민이 실제로 방문하거나 예약할 수 있는 후보 프로그램을 구성합니다.

사용 데이터 예시

- 서울시 문화행사 정보
- 서울시 공공서비스예약 정보
- 서울시 노인복지관 프로그램 정보
- 서울시 청년공간 프로그램 정보

이 데이터는 다음 판단에 쓰입니다.

- 어떤 프로그램이 실제로 존재하는지
- 어느 자치구에 있는지
- 실내/실외 성격이 어떤지
- 무료인지, 예약이 필요한지
- 가족/부모님/혼자 가기 좋은지

## 2. 공공데이터포털(data.go.kr)

역할: 오늘 서울의 컨디션을 읽어 추천 강도를 조정합니다.

사용 데이터 예시

- 기상청 단기예보 조회서비스
- 에어코리아 대기오염정보 조회서비스

이 데이터는 다음 판단에 쓰입니다.

- 오늘은 실내 우선인지 야외 우선인지
- 장거리 이동이 부담스러운 날인지
- 가족/부모님 동행 시 체감 만족도가 높을지

## 왜 두 포털을 함께 쓰는가

열린데이터광장만 쓰면 “갈 수 있는 후보”는 알 수 있지만, 오늘 당장 가기 좋은지는 알기 어렵습니다.

공공데이터포털만 쓰면 오늘의 날씨와 대기질은 알 수 있지만, 서울 시민이 실제로 갈 수 있는 공공 프로그램 후보는 부족합니다.

`오늘서울`은 이 둘을 합쳐서 아래 질문에 답합니다.

> 오늘 서울에서, 지금 조건으로, 실제로 갈 만한 공공 프로그램은 무엇인가?
